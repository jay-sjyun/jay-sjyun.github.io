ABOUT

LANGUAGE

  • HTML :
    웹 표준을 준수하며 검색 최적화를 위한 시맨틱 마크업 및 사용자 편의성을 위한 접근성을 고려합니다.
  • CSS :
    코딩 컨벤션을 준수하고 미디어 쿼리를 활용하여 반응형 웹을 구현합니다. 더불어 전처리기(SCSS)를 활용하여 가독성 및 유지보수 편의성을 향상합니다.
  • Javascript :
    원격 API를 호출하여 데이터를 전달받고 화면 요소를 동적으로 조작합니다. 또한 생산성 향상을 위해 jQuery 및 라이브러리를 유연하게 활용합니다.

TOOLS

  • Visual Studio Code :
    다양한 확장 기능을 활용하여 생산성 높은 개발 환경을 구축합니다.
  • Github :
    버전 관리를 통해 유지보수 편의성을 향상하고 코드의 충돌을 최소화하여 개발자와 협업 및 분업을 원활하게 수행합니다.
  • Figma & Adobe XD :
    레이아웃 구성에 최적화된 프로그램을 사용하여 웹 디자인 및 퍼블리싱을 효율적으로 수행합니다. 팀원들과 편리하게 피드백을 주고받으며 동시 작업 또한 가능합니다.
  • Photoshop & Illustrator :
    아이콘 제작 및 이미지 보정 등, 그래픽 디자인을 구현합니다.
